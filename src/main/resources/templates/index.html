Описание проекта

Проект выполнен в учебных целях для освоения возможностей Spring.
Проект является сервисом, предоставляющим информацию об автонарушителях. В системе имеются две роли: пользователи и автоинспекторы.

Пользователь добавляет описание автонарушения:
  • Адрес;
  • Номер машины;
  • Описание нарушения;
  • Фотографию нарушения.
У заявки есть статус:
  • Принята
  • Отклонена
  • Завершена
Стек технологий:
  • Java 17
  • PostgreSQL 14
  • Maven 3.8.5
  • Spring boot 2.7.6
  • Liquibase 4.15.0
  • Hibernate
  • h2Database 2.1.214
  • Bootstrap 5.2.3
Требования к окружению:
  • Java 17
  • PostgreSQL 14
  • Maven 3.8.5
