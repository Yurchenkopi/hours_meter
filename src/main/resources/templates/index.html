Описание проекта

Проект выполнен в учебных целях для освоения возможностей Spring.
Проект является сервисом, позволяющим контролировать дополнительное время, отработанное сотрудниками. В системе имеются две роли пользователей: сотрудники и работодатели.

Сотрудник добавляет временной интервал. При добавлении указываются следующие параметры:
  • Время начала работы;
  • Время окончания работы;
  • Был ли обеденный перерыв;
  • Учитывать ли в расчете основное рабочее время.

После добавления дополнительных трудодней сотрудник может сформировать отчет за выбранный период времени.

У работодателя есть возможность:
  • Просмотра дополнительно отработанного времени любого из сотрудников;
  • Простановки флага "не согласовано" для произвольного временного интервала;
  • Провести расчет заработанной платы за дополнительно отработанное время.
Стек технологий:
  • Java 21
  • PostgreSQL 14
  • Maven 3.8.5
  • Spring boot 2.7.6
  • Liquibase 4.15.0
  • Hibernate
  • Bootstrap 5.2.3
Требования к окружению:
  • Java 21
  • PostgreSQL 14
